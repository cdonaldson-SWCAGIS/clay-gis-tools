# Update Layer Form Default Values

**Quick Start**
1. Select a web map and click "Load Layers"
2. Check "Apply" for layers to update
3. Select field and enter default value
4. Click "Apply Changes"

**Form Sources**
Forms can be saved in two places:
- **Webmap**: Form is saved in the web map configuration
- **Layer**: Form is saved on the layer item itself
- **None**: No form exists for this layer

When a layer has a form on the layer item but not in the webmap, the form will be automatically copied to the webmap when you apply changes.

**Notes**
- Only layers with a form (Webmap or Layer) can be updated
- Different layers can use different field names
- The field dropdown shows available fields per layer

**Save as Copy**
- Creates a new web map with your changes
- Original remains unchanged
- Configure default suffix in Settings
